## О модуле
YTMusic — модуль для музыкальных сценариев через YouTube: поиск, выгрузка в чат и построение персональных плейлистов в БД.

## Как это работает
- Находит треки по текстовому запросу.
- Отправляет аудио в чат.
- Управляет локальными БД-плейлистами.
- Делает перенос библиотек через импорт/экспорт.
