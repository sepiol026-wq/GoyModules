# GoyPulse V9 🧠

[English Version](README_en.md)

## Описание
GoyPulse — это продвинутый модуль для Telegram-юзерботов (Friendly-Telegram и др.), основанный на цепях Маркова. Он позволяет боту обучаться на истории сообщений в чатах и имитировать общение пользователя, поддерживая контекст и стиль.

## Основные возможности
*   **Обучение (Training)**: Сбор и анализ сообщений для построения лингвистической модели.
*   **Автоответчик**: Интеллектуальная генерация ответов с учетом контекста последних сообщений.
*   **Скрытый режим (.gph)**: Выполнение команд в других чатах без их отображения пользователям (результаты приходят в лог-канал).
*   **Безопасность**:
    *   **GPB2 Backup**: Шифрование бэкапов базы данных с использованием Ed25519 (X25519) и AEAD.
    *   **Подпись обновлений**: Все официальные обновления подписаны цифровой подписью.
    *   **Self-Integrity Check**: Защита от модификации кода.
*   **Логирование**: Отдельный канал для системных событий, ошибок и отчетов.

## Установка
Отправьте в любой чат:
`.dlm https://raw.githubusercontent.com/sepiol026-wq/goypulse/main/goypulse.py`

## Команды
*   `.gpulse [on/off]` — Включить/выключить бота в чате.
*   `.gpref` — Запустить обучение в текущем чате.
*   `.gpstat` — Статистика базы и текущего чата.
*   `.gpinfo` — Анализ "вайба" чата и активности пользователей.
*   `.gpupdate` — Проверка и применение обновлений.
*   `.gpbackup` — Управление резервными копиями (шифрование, экспорт).
